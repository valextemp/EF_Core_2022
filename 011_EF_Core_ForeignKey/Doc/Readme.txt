
-- Отношения между сущностями
	-- Внешние ключи и навигационные свойства
		Для связей между сущностями в Entity Framework Core применяются внешние ключи и навигационные свойства.

	-- Настройка внешнего ключа через аннотации данных и Fluent API 
		- Настройка ключа с помощью аннотаций данных
			Чтобы установить свойство в качестве внешнего ключа, применяется атрибут [ForeignKey]:
				public class User
				{
					public int Id { get; set; }
					public string? Name { get; set; }
					public int? CompanyInfoKey { get; set; }
					[ForeignKey("CompanyInfoKey")]
					public Company? Company { get; set; }
				}

		- Настройка ключа с помощью Fluent API
			Для настройки отношений между моделями с помощью Fluent API применяются специальные методы: HasOne / HasMany / WithOne / WithMany. Методы HasOne и HasMany устанавливают 
			навигационное свойство для сущности, для которой производится конфигурация. Далее могут идти вызовы методов WithOne и WithMany, который идентифицируют навигационное 
			свойство на стороне связанной сущности. Методы HasOne/WithOne применяются для обычного навигационного свойства, представляющего одиночный объект,
